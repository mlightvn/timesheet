## About 勤怠システム

+ 勤怠管理
+ ユーザ、プロジェクト、タスク、部署なども追加、修正しやすい
+ AJAXを使っていますのでコードを見やすいし、パフォーマンスも早いです。
+ マウスで入力しやすい
+ エクセルでダウンロード
+ 言語変更：英語、日本語
